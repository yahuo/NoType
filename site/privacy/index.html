Privacy

隐私说明

NoType 本身是一个本地运行的 macOS 菜单栏应用，但它会把音频或文本发送给你自行配置的第三方服务，因此需要你了解数据流向。

本地权限

  • `Microphone`：用于采集录音音频
  • `Accessibility`：用于把文本粘贴回当前输入框

本地存储

  • `Access Token` 和 API key 等敏感凭证保存在 macOS Keychain
  • 其他设置保存在本地 `UserDefaults`

第三方服务

NoType 不自带语音识别服务和 AI 服务。你需要自行配置并承担这些服务的数据处理结果。

  • 语音识别：当前主链路使用 Doubao Streaming ASR
  • AI Rewrite：可选，支持 OpenAI-compatible 或 Gemini provider

是否发送数据、发送到哪里、使用哪种模型，都由你的本地设置决定。

建议

如果你准备把 NoType 发给其他用户使用，建议在 GitHub Releases 和官网同时明确说明两件事：当前构建未经过 Apple notarization，且应用本地运行时会把语音和改写请求发送到用户自己配置的服务端。
